# Overview

This directory contains the files for generating the documentation for Querqy. Querqy has plugins for the Elasticsearch, OpenSearch and Solr search engines. Search-engine specific content is put into tabs. Content that is the same for Elasticsearch and OpenSearch is put into a single tab labelled 'Elasticsearch/OpenSearch'.

The index.rst file contains installation instructions. When updating them for a new version Querqy, get the available Querqy versions from Maven Central and also provide download links to Maven Central only.
